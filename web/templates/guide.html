Guide d'Utilisation de TSAR Toolbox

Introduction

Bienvenue dans TSAR Toolbox, votre plateforme centralisée pour les tests d'intrusion. Ce guide vous expliquera comment utiliser les différentes fonctionnalités de l'outil.

La Méthodologie PTES

TSAR est structuré selon le Penetration Testing Execution Standard (PTES), une méthodologie reconnue qui divise un pentest en plusieurs phases logiques. Les modules de la toolbox sont organisés pour suivre ce flux de travail.

  • Phase 2: Intelligence Gathering - Collecte d'informations sur la cible.
  • Phase 4: Vulnerability Analysis - Scan actif pour trouver des failles.
  • Phase 5: Exploitation - Tentative d'exploitation des failles.
  • Phase 6: Post-Exploitation - Actions après avoir obtenu un accès.
  • Phase 7: Reporting - Génération du rapport final.

Comment utiliser un module ?

  1. Naviguez vers la page Modules depuis la barre latérale.
  2. Choisissez un module correspondant à la phase de votre pentest.
  3. Cliquez sur "Lancer".
  4. Remplissez les champs requis dans le formulaire (cible, mode, etc.).
  5. Cliquez sur "Lancer le Scan". La tâche sera exécutée en arrière-plan.
  6. Une fois le scan terminé, vous serez redirigé vers la page Rapports où vous pourrez télécharger le PDF chiffré contenant les résultats.

Analyse de CVE

La page Analyse CVE vous permet de transformer un simple identifiant de vulnérabilité (ex: CVE-2021-44228) en un scénario d'attaque complet, mappé sur le framework MITRE ATT&CK. C'est un outil puissant pour comprendre l'impact réel d'une faille.

Tâches Planifiées

Automatisez vos scans de routine. Vous pouvez planifier des scans quotidiens, hebdomadaires ou mensuels sur des cibles spécifiques. Les rapports générés apparaîtront automatiquement dans votre section Rapports.
